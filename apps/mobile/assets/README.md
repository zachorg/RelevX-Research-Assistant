# Assets

This directory should contain the app icons and splash screens for the mobile app.

## Required Assets

You can generate these assets using `npx expo-cli generate-icon-assets` or manually create them:

- `icon.png` - App icon (1024x1024px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `splash.png` - Splash screen (1242x2436px or similar)
- `favicon.png` - Web favicon (48x48px)

## Quick Setup

For now, you can use simple placeholder images or generate them from a single image:

```bash
# From this directory
npx expo start
# Then follow the prompts to generate icons if needed
```

Or create simple colored placeholder images temporarily until you have your branding ready.
